Carpeta `app/api/routes/`

Qué irá aquí:
- Archivos por grupo de endpoints, por ejemplo `chat_routes.py` para endpoints relacionados al chat.
- Cada archivo debe definir routers (`APIRouter`) y modelos de request/response si aplica.

Propósito:
- Organizar endpoints por feature y facilitar el montaje de routers en `main.py`.
